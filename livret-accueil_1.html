01 · Arrivée & départ

Check-in & check-out

Arrivée (check-in)
  • Garez-vous sur la place de parking privée réservée à votre logement (voir panneau).
  • Dirigez-vous vers la porte d'entrée de l'appartement.
  • Saisissez votre code PIN sur la serrure connectée Nuki — le code vous a été communiqué par message avant votre arrivée.
  • La porte s'ouvre automatiquement. Bienvenue !

Arrivée possible à partir de 15h00. Arrivée tardive ? Pas de problème, la serrure Nuki fonctionne 24h/24.

Départ (check-out)
  • Déposez les draps et serviettes utilisés dans le bac prévu à cet effet.
  • Rincez la vaisselle et laissez le logement dans un état correct.
  • Vérifiez que toutes les fenêtres sont fermées.
  • Refermez simplement la porte derrière vous — elle se verrouille automatiquement.

Départ avant 11h00. Merci de laisser les clés (si remises) sur le comptoir de la cuisine.
